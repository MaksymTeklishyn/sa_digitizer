\doxysubsection{Detailed Description}
A class representing a magnetic field, inheriting from \mbox{\hyperlink{classVectorField}{Vector\+Field}}. 

The \mbox{\hyperlink{classMagneticField}{Magnetic\+Field}} class allows for the definition and manipulation of a magnetic field based on a user-\/defined function. It provides methods to get the field strength at a specific position and to visualize the field component along a given direction.
